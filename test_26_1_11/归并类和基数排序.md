## 5）归并类（Merge Sort）

### 5.1 Merge（必会：两个有序段归并）

> “核心操作：把数组内的两个有序序列归并为一个”
>  Merge 后 A[low..high] 整体有序

```C
void Merge(int A[], int B[], int low, int mid, int high){
    // 复制到辅助数组B
    for(int i=low;i<=high;i++) B[i]=A[i];

    int i=low, j=mid+1, k=low;
    while(i<=mid && j<=high){
        if(B[i] <= B[j]) A[k++]=B[i++];   // <= 保稳定
        else             A[k++]=B[j++];
    }
    while(i<=mid)  A[k++]=B[i++];
    while(j<=high) A[k++]=B[j++];
}
```

### 5.2 递归归并排序（会写）

```C
void MergeSort(int A[], int B[], int low, int high){
    if(low>=high) return;
    int mid=(low+high)/2;
    MergeSort(A,B,low,mid);
    MergeSort(A,B,mid+1,high);
    Merge(A,B,low,mid,high);
}
```

## 6）非比较类：基数排序（Radix Sort，LSD）

> 初始化 r 个队列；对每一位依次“分配”“收集” 

```C
// 简化版：LSD 基数排序（仅演示整数非负，base=10）
void RadixSort(int A[], int n){
    int tmp[n];
    int exp=1;               // 1,10,100...
    int maxv=0;
    for(int i=0;i<n;i++) if(A[i]>maxv) maxv=A[i];

    while(maxv/exp>0){
        int cnt[10]={0};     // 计数=桶大小
        for(int i=0;i<n;i++) cnt[(A[i]/exp)%10]++;

        // 前缀和 -> 稳定收集位置
        for(int i=1;i<10;i++) cnt[i]+=cnt[i-1];

        for(int i=n-1;i>=0;i--){
            int d=(A[i]/exp)%10;
            tmp[--cnt[d]]=A[i];
        }
        for(int i=0;i<n;i++) A[i]=tmp[i];
        exp*=10;
    }
}
```
